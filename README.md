# Code-Kingdoms

Monorepo con Laravel Sail (backend) y Vue (frontend).

## Estructura
- `/backend` - API Laravel con MariaDB
- `/frontend` - App Vue

## Requisitos
- Docker Desktop
- Node.js 18+
- Composer

## Pasos para ejecutar el proyecto

1. **Dar permisos al script (importante)**
```bash
chmod +x start.sh

2. **Ejecutarlo**
```bash
./start.sh
